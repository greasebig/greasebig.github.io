Questions

Is greasebig free to use? +

Yes, greasebig is completely free. Create a project, share the link, and start building — no sign-up or payment required.

Do I need to create an account? +

No accounts needed. Just create a repo, share the link with collaborators, and start building — it's that simple.

Is my data stored or shared? +

We built greasebig to be open. All code is public, no accounts are needed, and the space is just for the builders. Code is stored permanently on GitHub. Once a project is published, it's accessible to everyone.

Can others see what I originally typed? +

Your collaborators see the final project, not your draft process. No signups, no subscriptions, no tracking. A tool built to help researchers be kinder to each other — nothing more.
