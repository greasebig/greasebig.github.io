📊 Visitor Tracking Methods

How to add analytics to your GitHub Pages site or GitHub Profile README

🔬

Professional Analytics

Google Analytics or Baidu Tongji — the most thorough approach. Track IP-derived geolocation, visit duration, device type, and more.

  1. Register a Google Analytics or 百度统计 account
  2. Add your GitHub Pages URL to get a JavaScript tracking snippet
  3. Paste the snippet into every HTML <head> tag and push
✅ IP → geolocation, duration, device, browser
⚠️ Requires account setup & GDPR notice
🗺️

Lightweight IP Counter

Quick & easy — embed a visitor map or counter from services like FreeCounterStat, RevolverMaps, or ClustrMaps.

  1. Visit freecounterstat.com or revolvermaps.com
  2. Configure your site URL, get an HTML/JS embed code
  3. Paste into your site — done!
✅ Zero config, instant visitor globe/counter
⚠️ Less detailed than professional analytics
👤

GitHub Profile Tracking

For your GitHub profile README — no JS allowed, so use image-based tracking. The browser loads the image and the server captures the IP.

  1. Get a Markdown image tag from RevolverMaps or Visitor Badge
  2. Paste ![visitors](https://ids.revolvermaps.com/...) into README.md
  3. A globe appears on your profile showing visitor locations
✅ Works on GitHub README (no JS needed)
⚠️ No duration/interaction tracking

📋 Quick Embed Snippets

RevolverMaps Globe (for HTML pages):
<script type="text/javascript" src="https://rf.revolvermaps.com/0/0/8.js?i=YOUR_ID&m=0&c=ff0000&cr1=ffffff&f=arial&l=33"></script>
Visitor Badge (for GitHub README):
![Visitors](https://api.visitorbadge.io/api/visitors?path=https%3A%2F%2Fgithub.com%2Fgreasebig&label=VISITORS&countColor=%236366f1)
Google Analytics gtag.js (replace GA_MEASUREMENT_ID):
<script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'GA_MEASUREMENT_ID');
</script>
